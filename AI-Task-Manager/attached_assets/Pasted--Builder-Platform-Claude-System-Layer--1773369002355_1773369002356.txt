בנוסף ל-Builder Platform, אני רוצה שתתכנן ותבנה גם שכבת Claude System Layer מלאה בתוך המערכת.

חשוב מאוד:
אני לא רוצה רק חיבור API קטן ל-Claude.
אני לא רוצה רק "צ'אט AI".
אני לא רוצה רק endpoint אחד כללי.

אני רוצה תשתית Claude ארגונית, מודולרית, מסודרת, metadata-aware, בטוחה, ועתידית, כך ש-Claude יוכל בעתיד לעבוד בתוך המערכת בצורה אמיתית, דרך APIs פנימיים נפרדים לפי אחריות ותפקיד.
