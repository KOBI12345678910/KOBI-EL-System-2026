1) מודלי שווי וניתוח פיננסי

זה הבסיס של sell-side, buy-side, private markets ו‑investment banking:

DCF / FCFF:
FCFF = EBIT*(1-T) + D&A - Capex - ΔNWC
EV = Σ FCFF_t/(1+WACC)^t + TV/(1+WACC)^n

FCFE:
FCFE = NI + D&A - Capex - ΔNWC + Net Borrowing

WACC:
WACC = (E/V)*Re + (D/V)*Rd*(1-T)

DDM / Gordon Growth:
P0 = D1 / (r-g)

Residual Income / EVA:
RI = NI - r*BV_(t-1) ;
Value = BV0 + PV(residual income)

Relative valuation: Comparable Companies, Precedent Transactions

SOTP: פירוק קונגלומרט לחלקים

LBO model

M&A Accretion/Dilution

APV: ערך בלי מינוף + ערך מגן המס
אלה בדיוק סוגי המודלים ש‑CFA Institute מגדירה כ‑present value, multiplier ו‑asset-based valuation.

2) כל המכפילים שבאמת עובדים איתם

המכפילים מתחלקים ל‑equity multiples ול‑enterprise multiples, והכלל הקריטי הוא להתאים בין המכנה למי שהמונה שייך לו: equity עם equity, enterprise עם firm. זה גם דגש מרכזי אצל Damodaran וגם אצל CFA.

המכפילים המרכזיים:

P/E = Price / EPS

Forward P/E = Price / NTM EPS

PEG = (P/E) / Growth

P/B = Market Cap / Book Equity

P/TBV = Price / Tangible Book

P/S = Market Cap / Sales

P/FCF = Equity Value / Free Cash Flow to Equity

Dividend Yield = Dividend / Price

EV/Sales

EV/EBITDA

EV/EBIT

EV/NOPAT

EV/FCFF

Net Debt / EBITDA

Interest Coverage = EBIT / Interest Expense

מכפילים סקטוריאליים חשובים:

בנקים: P/TBV, P/B, P/E, ROTCE/ROE

ביטוח: P/B, P/E, combined ratio, ROE

REITs / נדל"ן: P/FFO, EV/NOI, cap rate

SaaS: EV/ARR, EV/Revenue, Rule of 40, CAC payback, NRR

אנרגיה/כרייה: EV/EBITDAX, NAV, EV/boe

אשראי: spread, OAS, CDS spread, leverage ratios.
