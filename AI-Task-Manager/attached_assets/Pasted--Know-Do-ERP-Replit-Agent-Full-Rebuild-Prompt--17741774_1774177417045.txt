# Know Do ERP — Replit Agent Full Rebuild Prompt
# טכנו-כל עוזי — פרומפט מלא לבנייה מחדש

---

## הוראות כלליות ל-Agent

אתה בונה מערכת ERP מלאה בשם **Know Do** עבור חברת **טכנו-כל עוזי** — מפעל מסגרות ברזל, אלומיניום, נירוסטה וזכוכית.

### כללי ברזל:
- **Stack**: React + Node.js + PostgreSQL
- **שפה**: עברית RTL בלבד בכל ה-UI
- **Theme**: Dark Slate — רקע #0f172a, כרטיסים #1e293b, accent #38bdf8
- **מע"מ**: תמיד 18% (מכפיל 1.18)
- **מטבע**: ₪ — ערכים כספיים נשמרים כ-integer באגורות (agorot)
- **TypeScript strict mode** בכל הקוד
- **כל טבלה חייבת**: id, created_at, updated_at, created_by, is_active, notes
- **Audit log**: כל שינוי נרשם עם user, timestamp, action, old_value, new_value
- **Soft delete**: אף רשומה לא נמחקת — רק is_active = false
- **כל דף חייב לעבוד**: טעינה, יצירה, עריכה, מחיקה, חיפוש, סינון, ייצוא
- **Navigation**: סיידבר ימני קבוע עם כל הקטגוריות והתתי-קטגוריות

### פאזות בנייה:
1. **Phase 1**: DB Schema — בנה את כל הטבלאות עם כל השדות
2. **Phase 2**: API Routes — CRUD לכל ישות
3. **Phase 3**: UI Pages — כל הדפים עם טפסים, טבלאות, דשבורדים
4. **Phase 4**: אינטגרציות, אוטומציות, דוחות מתקדמים
