# ERP Module: Raw Materials

This directory contains the database setup and initial seed data for the Raw Materials module in the ERP system.

## Files

- `raw_materials_db_setup.sql`: Contains the SQL script to create tables, triggers, and seed data.

## Setup Instructions

1. Run the `raw_materials_db_setup.sql` script in your PostgreSQL database to create the tables and seed data.
2. Use the provided queries for testing and further development.
