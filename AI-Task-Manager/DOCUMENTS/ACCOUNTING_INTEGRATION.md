# מדריך ייבוא CSV לחשבשבת / פריורטי

## מטרת המסמך

מסמך זה מסביר כיצד לייצא נתונים פיננסיים ממערכת ה-ERP של טכנו-כל עוזי ולייבא אותם לתוכנת הנהלת חשבונות ישראלית (חשבשבת / פריורטי).

---

## שלב 1: ייצוא הנתונים מהמערכת

### גישה לדף הייצוא

1. התחבר למערכת ERP
2. נווט ל: **כספים ← ייצוא חשבונאי לחשבשבת**  
   (או ישירות בכתובת: `/finance/accounting-export`)

### בחירת תקופה וסוג ייצוא

1. בחר **חודש** ו**שנה** לייצוא
2. לחץ **רענן** — המערכת תציג:
   - ספירת רשומות לכל קטגוריה
   - **אזהרות** על נתונים חסרים (מספר עוסק, מספר חשבונית וכו')
3. בחר **סוג ייצוא**:
   - **ייצוא מאוחד** (מומלץ) — כל הנתונים בקובץ אחד
   - חשבוניות לקוחות בלבד
   - תשלומים שהתקבלו בלבד
   - הוצאות ספקים בלבד
4. לחץ **הורד קובץ CSV**

---

## שלב 2: ייבוא בחשבשבת

### ייבוא חשבוניות מכירה

1. פתח **חשבשבת**
2. עבור לתפריט: **קבצים ← ייבוא ← ייבוא תנועות**
3. בחר סוג מסמך: **חשבונית מכירה**
4. בחר את קובץ ה-CSV שהורד
5. מפה את העמודות לפי הטבלה הבאה:

| עמודה בקובץ              | שדה בחשבשבת              |
|---------------------------|---------------------------|
| מספר חשבונית              | מספר מסמך                |
| תאריך חשבונית             | תאריך                    |
| שם לקוח                   | שם לקוח                  |
| מספר עוסק/ח.פ לקוח        | מספר עוסק                |
| סכום לפני מע"מ            | סכום לפני מע"מ           |
| סכום מע"מ                 | סכום מע"מ                |
| סכום כולל מע"מ            | סכום כולל                |

### ייבוא חשבוניות קנייה (הוצאות ספקים)

1. עבור לתפריט: **קבצים ← ייבוא ← ייבוא תנועות**
2. בחר סוג מסמך: **חשבונית קנייה**
3. בחר קובץ הוצאות ספקים
4. מיפוי עמודות:

| עמודה בקובץ              | שדה בחשבשבת              |
|---------------------------|---------------------------|
| מספר חשבונית              | מספר מסמך                |
| תאריך חשבונית             | תאריך                    |
| שם ספק                    | שם ספק                   |
| מספר עוסק ספק             | מספר עוסק                |
| סוג מסמך                  | סוג מסמך                 |
| סכום לפני מע"מ            | סכום לפני מע"מ           |
| שיעור מע"מ %              | שיעור מע"מ               |
| סכום מע"מ                 | סכום מע"מ                |
| סכום כולל מע"מ            | סכום כולל                |

---

## שלב 2 (חלופה): ייבוא בפריורטי

1. פתח **פריורטי (Priority)**
2. עבור ל: **ממשקי ייבוא ← ייבוא קבצי CSV**
3. בחר תבנית: **חשבוניות / קבלות**
4. העלה את קובץ ה-CSV
5. מפה שדות לפי הכותרות

---

## פורמט הקובץ

### כותרות הקובץ (ייצוא מאוחד)

```
סוג רשומה, מספר מסמך, תאריך, שם, מספר עוסק, סכום לפני מע"מ, מע"מ, סה"כ כולל מע"מ, סטטוס, אמצעי תשלום, אסמכתא, הערות
```

### ערכי "סוג רשומה"

| ערך              | משמעות                 |
|-------------------|------------------------|
| חשבונית לקוח      | חשבונית מכירה          |
| תשלום מלקוח       | קבלה / אישור תשלום     |
| חשבונית ספק       | חשבונית קנייה          |

### פורמט תאריך

`YYYY-MM-DD` (למשל: `2025-03-15`)

### פורמט מספרים

מספרים עשרוניים עם נקודה (לא פסיק) כמפריד. לדוגמה: `1234.50`

### קידוד תווים

UTF-8 עם BOM — מבטיח תצוגה תקינה של עברית ב-Excel וחשבשבת

## אזהרות נפוצות ופתרונן

### "מספר עוסק חסר"
**בעיה:** לקוח או ספק ללא מספר עוסק מורשה  
**פתרון:** עדכן את פרטי הלקוח/ספק במערכת ERP לפני הייצוא  
**מיקום:** מכירות ← לקוחות ← ערוך לקוח ← שדה "מספר עוסק"

### "מספר חשבונית חסר"
**בעיה:** חשבונית ללא מספר סידורי  
**פתרון:** וודא שכל החשבוניות קיבלו מספר ידנית או אוטומטית

### תווים לא תקינים בקובץ
**בעיה:** Excel מציג שאלונים במקום עברית  
**פתרון:** פתח את Excel ← ייבוא נתונים ← בחר UTF-8 עם BOM

### סכומים שגויים
**בעיה:** מע"מ לא מחושב כ-17%  
**פתרון:** המערכת מייצאת את הנתונים כפי שנרשמו. בדוק את הרישום המקורי

---

## דרישות רשות המיסים הישראלית

לפי תקנות מס ערך מוסף, כל חשבונית מס חייבת לכלול:

- מספר חשבונית סידורי ייחודי
- תאריך הוצאת החשבונית
- שם ומענו של העוסק המורשה (המוכר)
- מספר עוסק מורשה של המוכר
- שם הלקוח + מספר עוסק (אם לקוח עסקי)
- תיאור הטובין/שירות
- מחיר לפני מע"מ
- שיעור המע"מ וסכומו
- סכום כולל מע"מ

## שאלות נפוצות

**שאלה:** האם ניתן לייצא מספר חודשים ביחד?  
**תשובה:** לא — יש לייצא חודש אחד בכל פעם.

**שאלה:** מה קורה אם חשבונית בוטלה?  
**תשובה:** חשבוניות מבוטלות (status = cancelled) אינן נכללות בייצוא.

**שאלה:** מה שיעור המע"מ הנכון?  
**תשובה:** 17% החל מינואר 2025. המערכת מייצאת את השיעור כפי שנרשם בחשבונית.

**שאלה:** האם הייצוא כולל חשבוניות דרישה (חשבוניות שאינן מס)?  
**תשובה:** כן — כל סוגי המסמכים כלולים. יש לסנן ידנית בחשבשבת אם נדרש.
