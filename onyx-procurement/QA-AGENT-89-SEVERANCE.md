## 4. הסבר מקצועי — מה המערכת בפועל עושה?

המערכת `payroll-autonomous` מבצעת פעולה **אחת בלבד** הקשורה לפיצויים:

```javascript
// שורה 11
const SEV = 0.0833;

// שורה 61 — בתוך autoCalcEmployee
const sev = gross * SEV;

// שורה 64 — נצבר לעלות מעסיק
const erCost = gross + penR + sev + bl*1.05;
```

זוהי **הפרשה חודשית צפויה לקרן פיצויים** בשיעור 8.33% (≈1/12), שנכללת בעלות המעסיק בתלוש. זו התנהגות תקינה לחישוב **שוטף חודשי**, אבל המערכת **אינה יודעת**:

1. האם ההפרשה הזו משולמת בפועל לקרן פנסיה, קופת גמל, או "מופרדת בחברה".
2. האם העובד תחת סעיף 14 (ולכן ההפרשה **היא** הפיצויים הסופיים).
3. מה קורה **בסיום העסקה** — אין module, אין function, אין UI.
4. האם יש השלמה נדרשת, מס לתשלום, או טופס להוצאה.

בפועל, לצורכי דרישות Dimension "פיצויי פיטורים" — **המערכת עונה על 0 מ-8 דרישות**.
