## המלצות מיידיות לפי סדר עדיפות

1. **להוסיף תרגום שגיאות לעברית + retry** ב-`api()` — קריטי לקובי.
2. **להוסיף `confirm()` לפעולות הרסניות** (שליחת RFQ, אישור, החלטת AI) — מונע אסונות.
3. **לתקן ניגודיות**: `#64748b` → `#94a3b8`, `#475569` → `#cbd5e1`.
4. **Media query למסך נייד**: `@media (max-width:640px) { grid-template-columns: 1fr; }`.
5. **להוסיף focus-visible** לכל הכפתורים והשדות.
6. **להוסיף loading state** לכל mutation, לא רק ל-RFQ send.
7. **להוסיף empty state** לספקים/הצעות/קבלנים.
8. **Onboarding card** בדשבורד ריק — "👋 ברוך הבא! 4 צעדים להתחיל".
9. **תרגום סטטוסים** (`draft` → "טיוטה" וכו').
10. **Last updated timestamp** בheader + "נתונים לא עדכניים" אם API נכשל.
