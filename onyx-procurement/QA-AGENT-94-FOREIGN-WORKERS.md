## 3. דוגמת קוד בעייתית — חישוב עלות מעסיק

**מיקום:** `App.jsx` שורה 64
```js
const erCost = gross + penR + sev + bl*1.05;
```

**הבעיה:** חישוב זה תקף רק לעובד תושב. עבור עובד זר בענף הבניין:
```
erCost_real = gross
  + penR (לא תמיד רלוונטי)
  + sev (8.33%)
  + bl * ratio  (ביטוח לאומי — שיעורים שונים לעובד זר)
  + levy        (היטל 10-20% — חסר)
  + healthPrem  (ביטוח רפואי פרטי — חסר)
  + bankDeposit (18.5% בבנייה — חסר)
```

ההפרש יכול להגיע ל־**25–30%** בעלות המעסיק האמיתית, שמשמעותו הפסד ודאי על כל פרויקט שתומחר לפי המערכת הנוכחית.

## 4. דוגמת קוד בעייתית — ברירת מחדל של נקודות זיכוי

**מיקום:** `App.jsx` שורה 316 + 513 + 522
```js
setForm({baseSalary:"",creditPoints:"2.25",...})   // ברירת מחדל בטופס
...
const d={...form, creditPoints:+form.creditPoints||2.25, ...}  // ברירת מחדל בשמירה
```

**הבעיה:** 2.25 נקודות הוא ברירת המחדל האוטומטית לכל עובד. עובד זר **אינו זכאי** לנקודות זיכוי (למעט חריגים מאוד ספציפיים כמו עובד מסייע לצעיר חולה או עובד שהוא תושב פלסטיני במצבים מסוימים). הענקה אוטומטית של 2.25 נק' → ניכוי מס חסר של כ־`2.25 × 242 = 544.5 ₪` לחודש.

**אומדן חשיפה:** על 5 עובדים זרים במשך שנה = `5 × 544.5 × 12 = 32,670 ₪` ניכוי מס חסר — המעסיק חב בסכום זה ישירות לרשות המסים + קנסות.

## 5. המלצות תיקון (Remediation)

### 5.1 P0 — חייב מיידי (לפני כל שימוש בעובד זר)
1. **להוסיף שדה `workerType`** עם ערכים: `resident`, `foreign`, `infiltrator`, `palestinian`.
2. **לנטרל אוטומטית `creditPoints`** כאשר `workerType !== 'resident'` → להגדיר ברירת מחדל 0.
3. **להוסיף `permitExpiry` + `permitType`** עם ולידציה שחוסמת את הרצת השכר אם האשרה פגה.
4. **להרחיב את `taxCalc`** לקבל `workerType` ולהחיל מדרגות מיוחדות (למשל מס אחיד 25% מהשקל הראשון לעובד זר בענפים מסוימים).
5. **להוסיף חישוב `foreignWorkerLevy`** ולכלול אותו ב־`erCost`.
6. **להוסיף קבועים** `MIN_WAGE_MONTHLY=5880`, `MIN_WAGE_HOURLY=32.30`, `MAX_HOUSING_DEDUCT=420` וכו'.
7. **להוסיף אכיפת שכר מינימום לפני ניכויים** — `assertMinWage(gross - deductions)`.

### 5.2 P1 — נדרש בתוך רבעון
1. מודול דיווח PIBA (ייצוא CSV בפורמט הרשות).
2. מודול פיקדון בנק חיסכון (18.5%) עם מעקב הפקדות.
3. דשבורד התראות על אשרות שפגות תוך 30/60/90 יום.
4. מודול ביטוח רפואי פרטי — הגדרת ספקים, תקרה, ניכוי מתלוש.
5. תיעוד אוטומטי של חוזים בשפת העובד (רוסית/סינית/תאית/ערבית) — חובה חוקית.

### 5.3 המלצת תהליך עסקי (לקובי)
**בתקופת הביניים עד התיקון — אל תשתמש במערכת זו עבור עובדים זרים.**
עדיף לנהל אותם ב־Excel נפרד או מול רו"ח חיצוני המתמחה בעובדים זרים בענף הבניין. הסיכון המשפטי הוא פלילי, לא רק אזרחי.
